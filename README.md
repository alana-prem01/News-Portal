# News Portal

News Portal developed as part of the PENOFT Internship Skill Test.

## Features

### Public Interface
- Home Page
- Category Page
- Single News Page

### Admin Panel
- Admin Login
- Create News
- Edit News
- Delete News
- Category Management

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## Installation

### Frontend
npm install
npm run dev

### Backend
npm install
npm start
