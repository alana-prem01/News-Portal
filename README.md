# News Portal

News Portal developed as part of the PENOFT Internship Skill Test.

Frontend: news-portal-mb5y9kqn6-alana-prem01s-projects.vercel.app

Backend API:  https://news-portal-gcq9.onrender.com

## Features

### Public Interface
- Home Page
- Category Page
- Single News Page

### Admin Panel
- Admin Login
- Create News
- Edit News
- Delete News
- Category Management

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## Installation

### Frontend
npm install
npm run dev

### Backend
npm install
